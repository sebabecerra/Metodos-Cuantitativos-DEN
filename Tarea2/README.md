# Tarea 2 - Métodos Cuantitativos

Esta carpeta contiene la entrega final de la Tarea 2.

## Archivos principales

- `Tarea_2_Resuelta.pdf`: informe final para entregar.
- `Tarea_2_Resuelta.tex`: fuente LaTeX del informe final.
- `Tarea 2 2026.pdf`: enunciado original de la tarea.
- `requirements.txt`: paquetes necesarios para ejecutar los notebooks.

## Carpetas

- `notebooks/`: notebooks ejecutables con el desarrollo detallado de cada pregunta y un notebook completo.
- `data/`: bases SPSS necesarias para reproducir los análisis.

## Notebooks

- `notebooks/Tarea_2_Completa.ipynb`: solución consolidada de las cuatro preguntas.
- `notebooks/Tarea_2_P1_Mediacion_Moderada.ipynb`: Pregunta 1, mediación moderada.
- `notebooks/Tarea_2_P2_Analisis_Factorial.ipynb`: Pregunta 2, análisis factorial exploratorio.
- `notebooks/Tarea_2_P3_SEM.ipynb`: Pregunta 3, CFA/SEM.
- `notebooks/Tarea_2_P4_MANOVA.ipynb`: Pregunta 4, MANOVA factorial.

Los notebooks buscan automáticamente las bases en `data/` o `../data/`, dependiendo de si se ejecutan desde la raíz de `Tarea2` o desde la carpeta `notebooks/`.
