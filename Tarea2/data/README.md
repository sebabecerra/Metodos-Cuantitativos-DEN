# Datos de Tarea 2

Esta carpeta contiene las bases SPSS necesarias para ejecutar los notebooks de la Tarea 2.

- `Med_Mod.sav`: Pregunta 1, mediación moderada.
- `EECS-R.sav`: Pregunta 2, análisis factorial exploratorio.
- `Laboral.sav`: Pregunta 3, CFA/SEM.
- `HBAT_200.sav`: Pregunta 4, MANOVA factorial.

Los notebooks buscan automáticamente esta carpeta `data` cuando se ejecutan desde `Tarea2` o desde la raíz del repositorio.
